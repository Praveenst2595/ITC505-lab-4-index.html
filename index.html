Addendum

  1. Font Choices: The fonts 'Lora' and 'Merriweather' were chosen for their readability and professional appearance. 'Lora' is used for body text to provide a clean, modern look, while 'Merriweather' is used for headings to create a clear hierarchy and draw attention to section titles.

  2. Sizes of Headings and Main Text: The main heading (<h1>) is set to 2.5em to ensure it stands out as the most important element on the page. Subheadings (<h2> and <h3>) are sized at 2em and 1.5em respectively to create a visual hierarchy, making the content easy to scan. The main text is set to 1.1em to enhance readability.

  3. Color Choices: The color palette is chosen for clarity and professionalism. The background color (#f9f9f9) provides a neutral backdrop, while the main text color (#333) ensures high contrast for readability. Accent colors like #1abc9c for the main heading, #3498db for subheadings, and #e74c3c for tertiary headings add visual interest without overwhelming the content.

  4. Margin and Padding Choices: Margins and padding are used strategically to create space between elements and enhance readability. The body has 20px padding to ensure the content does not touch the edges of the browser. Headings have padding and margin adjustments to separate them from the content that follows, making the document more visually appealing and easier to navigate.

  5. HTML Abuse: Originally, some HTML tags were used based on their default styling rather than their semantic meaning. For instance, an <h2> tag was used for the title of the essay because it looked better, even though it was the highest-level heading. This has been corrected to an <h1> tag to maintain proper document structure and accessibility.

  6. Footer Placement: The footer is placed at the bottom of the page using flexbox to ensure it remains at the bottom regardless of the content length. This approach keeps the layout consistent and professional.
